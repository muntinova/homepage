jQuery Quiz I

1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?

Answer:

    -the jQuery function
        • the dollar sign with the parentheses is the shorter name of the jQuery function. 

    -the jQuery object
        • the object returned by the jQuery function that often represents a group of elements
        • $ always (even for ID selectors) returns array-like object: jQuery object
        • returned jQuery object wraps originally selected DOM objects.
        • can access actual DOM object by accessing elements of jQuery object.

2. Write jQuery code to find all h1 elements that are children of a div element and make their background color red. 

       <body>
            <h1>abc</h1>
           <div>
                <h1>div-1</h1>
                <h1>div-2</h1>
            </div>
            <h1>xyz</h1>
        </body>

Answer:

     <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js" type="text/javascript"></script>
       <script>
            $(document).ready(function(){
                $("div h1").css("background-color","red");
            });
     </script>

3. Use a jQuery method to insert the text "YES!" at the end of the <p> element. 

       <!DOCTYPE html>
            <html>
                <head>
                    <script>
                        <!—INSERT YOUR JQUERY CODE HERE - - >
                    </script>
                </head>
                <body>
                    <p>Is jQuery FUN or what? </p>
            </body>
       </html>

Answer:

    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js" type="text/javascript"></script>
    <script>
        $(document).ready(function(){
            $("p").append("Yes");
        });
    </script>
